# Curso Gustavo Guanabara

Repositório com exercícios, anotações e projetos desenvolvidos a partir dos cursos do **Curso em Vídeo**, ministrados por Gustavo Guanabara.

## 📌 Objetivo

Organizar o conteúdo estudado durante os cursos, registrando a evolução prática em:

- Lógica de programação
- Desenvolvimento web
- Versionamento com Git/GitHub

## 🧠 Contexto

Os cursos do Curso em Vídeo são voltados para iniciantes e focam em aprendizado prático por meio de:

- Aulas teóricas
- Exercícios
- Desafios

Este repositório reúne a aplicação direta desses conteúdos.

## 🛠️ Tecnologias abordadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub
- (outras conforme avanço no curso)

## 📂 Estrutura do repositório

Os arquivos estão organizados conforme cursos ou módulos.
